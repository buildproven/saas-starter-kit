# Legal Templates

Production-ready legal templates for SaaS products. Find and replace all `[BRACKETED]` placeholders before publishing.

## Files

| File                  | Purpose                                               |
| --------------------- | ----------------------------------------------------- |
| `privacy-policy.md`   | GDPR/CCPA/CPRA compliant privacy policy               |
| `terms-of-service.md` | Full SaaS terms covering subscriptions, IP, liability |
| `cookie-policy.md`    | Cookie disclosure with table of specific cookies      |

## Required Placeholders

Search and replace these in all three files:

| Placeholder          | Example                              |
| -------------------- | ------------------------------------ |
| `[COMPANY_NAME]`     | Acme Corp LLC                        |
| `[COMPANY_TYPE]`     | a Delaware Limited Liability Company |
| `[WEBSITE_URL]`      | `https://acme.com`                   |
| `[PRODUCT_NAME]`     | Acme SaaS                            |
| `[PRIVACY_EMAIL]`    | `privacy@acme.com`                   |
| `[SUPPORT_EMAIL]`    | `support@acme.com`                   |
| `[LEGAL_EMAIL]`      | `legal@acme.com`                     |
| `[EFFECTIVE_DATE]`   | January 1, 2026                      |
| `[PHYSICAL_ADDRESS]` | 123 Main St, City, State ZIP         |

## Important Notes

- These are templates, not legal advice. Have a lawyer review before publishing.
- If you use services not listed (e.g., different analytics, email providers), update the provider tables.
- If you don't have a newsletter, remove the newsletter-specific sections.
- If you use AI features, the AI disclosure sections are pre-written for OpenAI/Anthropic — update as needed.
